🌐 Task Manager App

📌 Overview  
This is a Task Manager web application built using HTML, CSS, and JavaScript to help users manage their daily tasks in a simple and interactive way.

🛠️ Technologies Used  
HTML  
CSS  
JavaScript  

✨ Features  
- Add new tasks  
- Edit tasks  
- Delete tasks  
- Mark tasks as completed  
- Progress bar and task counter  
- Local Storage support  
- Responsive design  
- Confetti effect on task completion  

💡 Note  
This project was built for learning and improving my front-end development skills.
